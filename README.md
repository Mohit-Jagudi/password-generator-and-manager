# Password Generator and Manager

A simple and interactive **Python-based Password Generator and Manager** that allows users to generate strong passwords and securely store them using file handling.

---

## Features

* Generate secure and random passwords
* Custom password generation (A-Z, a-z, 0-9, symbols)
* Minimum password length validation
* Option to save passwords with app names
* Overwrite existing passwords (no duplicates)
* Fetch saved passwords using app name
* Case-insensitive search (INSTA = insta)
* File handling using `.txt` file
* Exception handling for smooth user experience
* Beginner-friendly CLI interface

---

## Technologies Used

* Python
* File Handling
* Random Module
* String Module
* Exception Handling

## How to Run

1. Download the project files (ZIP or manually)

2. Create a new folder on your system
   Example: `password-generator-and-manager`

3. Move all project files into that folder:

   * `password_manager.py`

4. Open this folder in any code editor (like VS Code)

5. Open terminal inside the editor

6. Run the program using:

```bash
python password_manager.py
```

7. Follow the instructions on screen

---

## How to Use

1. Choose option:

   * Check saved passwords
   * Generate new password

2. Enter required inputs:

   * Password length
   * Random or custom mode

3. Generate password

4. Save password with app name (optional)

5. Retrieve saved passwords anytime
